🔐 Password Generator
A simple, browser-based password generator that creates secure, customizable passwords instantly. No installs required.
🔗 Live Demo: https://g00440625.github.io/password-generator/

📌 Project Description
This project is a password generator web app that allows users to create strong, random passwords based on their own preferences.
Users can:
- Set the password length
- Toggle inclusion of uppercase letters, lowercase letters, numbers, and symbols
- View a password strength indicator (Weak / Medium / Strong)
- Generate a new password instantly with one click

The goal of this project is to help users create secure passwords that are hard to guess, reducing the risk of their accounts being hacked.

🛠️ Technologies Used
HTML – Structure and layout
CSS – Styling and responsive design
JavaScript – Password logic and DOM manipulation
Font Awesome via CDNJS – Icons for buttons (copy, refresh, etc.)
Git & GitHub – Version control and repository hosting
GitHub Pages – Live deployment

⚙️ Setup Instructions
Option 1 — Use the Live Demo
Just visit: https://g00440625.github.io/password-generator/
Option 2 — Run Locally
1. Clone the repository: git clone https://github.com/g00440625/password-generator.git
2. Navigate into the folder: cd password-generator
3. Open index.html in your browser
4. Optionally, use a Live Server in VS Code for real-time updates.

📖 Usage
1. Adjust the password length using the slider.
2. Check or uncheck the character options:
✅ Include Uppercase (A–Z)
✅ Include Lowercase (a–z)
✅ Include Numbers (0–9)
✅ Include Symbols (!@#$...)
3. Click "Generate Password"
4. The password appears in the output box.
5. Use the copy icon to copy it to your clipboard.
